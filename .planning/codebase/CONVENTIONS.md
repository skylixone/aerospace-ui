# Conventions

Design and development standards for Aerospace UI.

- **Visual Style:** 
  - Amber-on-black color scheme.
  - 0px corner radius for an industrial, rigid aesthetic.
  - Typography: Geist Mono.
  - Labels: All-caps for headers and UI labels.
- **CSS Architecture:**
  - Token-based design via `:root` CSS variables.
  - BEM-like component classes (e.g., `.btn`, `.tag`).
