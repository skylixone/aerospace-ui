# Architecture

The system uses a flat, root-based architecture optimized for direct execution.

- **Layout Structure:**
  - Sidebar (`kit-nav`): Navigation and component indexing.
  - Main (`kit-main`): Content display area.
  - Sections (`kit-section`): Content grouping and categorization.
- **Helper Scripts:**
  - `foundation_tune.js`: Core UI logic and initialization.
  - `nav_highlight.js`: Navigation state management and highlighting.
